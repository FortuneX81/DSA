\# Circular Linked List – Round Robin Scheduling



This program demonstrates a Circular Linked List.



In a circular linked list:

The last node points back to the first node.



This is used to simulate Round Robin Scheduling.



RoundRobinSimple:

\- Executes processes in circular order.



RoundRobinAdvanced:

\- Adds execution time to processes.

\- Simulates time-based scheduling.



The circular structure allows continuous looping through processes.
