\# Data Structures and Algorithms Project



This project demonstrates basic Data Structures and Algorithms concepts using Java.



The project is divided into two main sections:



\## 1. Array

This section contains an implementation of Matrix Multiplication using 2D arrays.

The program multiplies two matrices using three nested loops and prints the result.



Files included:

\- MatrixMultiplication.java

\- Pseudocode.txt

\- README.md



\## 2. Linked List

This section contains three types of linked lists:



\### Singly Linked List

Implements a Playlist system.

\- Simple Playlist: Stores and displays song names.

\- Advanced Playlist: Plays WAV audio files from the local computer.



\### Doubly Linked List

Implements Browser Navigation.

\- Simulates visiting pages.

\- Allows moving backward and forward.



\### Circular Linked List

Implements Round Robin Scheduling.

\- Simple version cycles through processes.

\- Advanced version simulates execution time for processes.



Each folder contains:

\- Java implementation

\- Pseudocode

\- README explanation



This project demonstrates understanding of:

\- Arrays

\- Matrix multiplication

\- Singly linked lists

\- Doubly linked lists

\- Circular linked lists

\- Basic algorithm logic
